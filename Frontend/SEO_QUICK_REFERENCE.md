## 📱 CRITICAL SEO ELEMENTS

### Must Have on Every Page:
```
✓ Unique title tag (30-60 chars)
✓ Unique meta description (50-160 chars)
✓ Canonical URL
✓ Robots meta (index, follow)
✓ H1 heading (1 per page)
✓ Open Graph tags
✓ Twitter card tags
✓ Structured data (JSON-LD)
```

### Optional but Recommended:
```
→ Alt text on all images
→ Internal links (3-5 per page)
→ Heading hierarchy (H1 → H2 → H3)
→ Meta keywords (if targeting specific terms)
→ Hreflang (for multi-language)
→ Schema markup enhancements
```

## ⚠️ SEO MISTAKES TO NEVER MAKE

### Critical Mistakes (Kills SEO):
❌ Duplicate content across pages
❌ Keyword stuffing in titles/descriptions
❌ Broken internal links
❌ Unoptimized images (too large)
❌ Redirect chains (A→B→C)
❌ No mobile optimization
❌ Slow page load (> 3 seconds)
❌ No SSL certificate (HTTPS)

### Common Mistakes (Hurts SEO):
⚠️ Inconsistent schema markup
⚠️ Missing alt text on images
⚠️ Thin or duplicate meta descriptions
⚠️ Poor heading hierarchy
⚠️ No internal linking strategy
⚠️ Ignored crawl errors
⚠️ Stale content (no updates)
⚠️ Over-optimized anchor text
